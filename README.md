# 本地网页浏览器

## 项目介绍
这是一个基于Electron框架开发的本地网页浏览器，具有简洁美观的界面设计和完整的浏览器功能。浏览器采用淡紫色主题，界面紧凑，所有功能按钮都集成在菜单栏中，提供了良好的用户体验。

## 功能特性
- 简洁美观的淡紫色主题界面
- 无边框设计，包含自定义窗口控制按钮（关闭、最小化、最大化）
- 完整的菜单栏功能：
  - 打开网址（Ctrl+O）
  - 前进/后退导航
  - 刷新/停止加载
  - 窗口控制（最大化/最小化/关闭）
  - 关于信息
- 快捷键支持
- 加载状态指示器
- 初始欢迎页面
- 修复了Ctrl+O快捷键报错问题

## 项目结构
```
sa_webfrontend/
├── index.js              # Electron主进程文件
├── index.html            # 渲染进程HTML文件
├── icon.svg              # 应用图标
├── package.json          # 项目配置文件
├── electron初始化语句.txt  # 初始化记录
└── README.md             # 项目说明文档
```

## 技术栈
- Electron ^39.2.4
- HTML/CSS/JavaScript
- 原生WebView组件

## 运行方式
1. 安装依赖：`pnpm install`
2. 启动应用：`pnpm start`

## 快捷键说明
- Ctrl+O：打开网址
- Ctrl+Right：前进
- Ctrl+Left：后退
- Ctrl+R：刷新
- Esc：停止加载
- F11：最大化/还原窗口
- Ctrl+M：最小化
- Ctrl+W：关闭窗口
- Ctrl+Q：退出应用

## 设计说明
- 主题色：淡紫色（#f0e6ff）
- 图标：自定义SVG图标，简洁明了
- 界面：紧凑设计，无边框，只显示核心内容
- 圆角：小半径圆角设计，保持简洁

## 开发说明
本项目采用Electron框架，前后端不分离，使用原生WebView组件显示网页内容。所有功能通过Electron的IPC通信机制实现主进程与渲染进程的交互。
